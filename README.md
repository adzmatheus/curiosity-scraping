Os cristãos ortodoxos são membros de um ramo do cristianismo que se separou da Igreja Católica romana em 1054 e não sofreu influência ocidental. Na Igreja Ortodoxa oriental, o casamento dos padres é aceito. Apenas os bispos devem manter o celibato. O grupo mais numeroso da comunidade ortodoxa é a Igreja Ortodoxa russa.
